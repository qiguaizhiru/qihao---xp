================================================
  iMouseXP 自动化工具 V2.0 - 使用说明
================================================

【环境要求】
  - Python 3.10 或以上版本（推荐 3.12）
  - iMouseXP 内核已启动，设备已连接
  - 安装 Python 时需勾选"添加到PATH"

【首次安装】
  1. 解压压缩包到任意目录
  2. 双击 install.bat 安装依赖
     - 如果电脑没有 Python，会自动安装 Python 3.12
     - 自动安装所有依赖包（无需联网）
     - 自动修复兼容性问题
  3. 双击 start.bat 启动工具
  4. 如果遇到编码错误，双击 fix_imouse.bat

【功能介绍】

  一、一键发布（TikTok 抖音国际版）
  ----------------------------------------
  发布内容设置：
    - 内容类型：选择图片或视频
    - 素材文件：选择单个图片或视频文件（所有设备发同一个）
    - 素材文件夹：选择一个文件夹，按自定义名分配素材（见下方说明）
    - 音乐URL：TikTok 音乐链接（可选，留空则不添加音乐）
    - 标题：帖子标题
    - 描述标签：#话题标签 和描述文字

  发布参数：
    - 步骤间隔(秒)：每个操作步骤之间的等待时间，默认 3 秒
    - 识别等待(秒)：图像识别的最长等待时间，默认 15 秒
    - 设备间隔(秒)：切换设备之间的等待时间，默认 2 秒

  发布方式：
    - 一键发布到所有在线设备：自动发布到所有已连接设备
    - 发布到选中设备：只发布到左侧勾选的设备
    - 停止发布：随时中断发布流程

  二、素材文件夹模式（按自定义名分配）
  ----------------------------------------
  当不同设备需要发布不同内容时，使用文件夹模式：

  文件夹结构示例：
    素材/
      设备A的自定义名/
        video1.mp4
      设备B的自定义名/
        photo1.jpg
        photo2.jpg
      设备C的自定义名/
        video2.mp4

  说明：
    - 父文件夹下的每个子文件夹名称对应设备的"自定义名"
    - 自定义名就是在 iMouseXP 控制台左侧设置的设备别名
    - 发布时自动匹配子文件夹名和设备自定义名
    - 没有匹配到子文件夹的设备会被跳过
    - 子文件夹内支持多层嵌套，会递归扫描所有媒体文件

  三、Excel 批量导入
  ----------------------------------------
  导入 .xlsx 文件，每行一条发布任务。
  表格列格式：
    devices | file | type | url | title | description | status | scheduled_time

  说明：
    - devices：设备名或自定义名
    - file：素材文件路径
    - type：picture 或 video
    - url：音乐URL（可选）
    - title：标题
    - description：描述标签
    - status：状态（留空）
    - scheduled_time：定时发布时间（留空=立即发布）

  参考文件：tiktok_data.xlsx

  四、上传功能
  ----------------------------------------
  将图片/视频上传到手机：

  上传目标：
    - 上传到相册：文件存入手机相册
    - 上传到文件系统：文件存入手机文件目录（需 iOS 15+）

  上传方式：
    - 素材文件夹模式：选择父文件夹，子文件夹名=设备自定义名
      每台设备只上传自己对应文件夹里的文件
    - 统一文件模式：选择文件或文件夹，所有设备上传相同文件
    - 上传到选中设备：只上传到勾选的设备
    - 一键上传到所有设备：上传到所有已连接设备

  五、下载功能
  ----------------------------------------
  从手机下载图片/视频到电脑：

  下载来源：
    - 从相册下载
    - 从文件系统下载（需 iOS 15+）

  六、浏览/管理
  ----------------------------------------
  查看手机上的文件：
    - 查看相册列表和相册内容
    - 查看文件系统目录

【发布流程（自动执行）】
  第 1 步：上传素材到手机相册
  第 2 步：按 HOME 键回到主屏幕
  第 3 步：通过图像识别找到并打开 TikTok
  第 4 步：点击 + 号按钮（创建内容）
  第 5 步：从相册选择刚上传的素材
  第 6 步：点击"下一步"
  第 7 步：添加音乐（如果填了音乐URL）
  第 8 步：通过剪贴板粘贴标题
  第 9 步：通过剪贴板粘贴描述/标签
  第10步：点击"发布"按钮

【icon 文件夹说明】
  图像识别用的模板图片：
  - tiktok.bmp            TikTok 应用图标
  - +black.bmp            深色 + 号按钮
  - +white.bmp            浅色 + 号按钮
  - next.bmp              下一步按钮
  - post.bmp              发布按钮
  - 旧版post.bmp          旧版发布按钮
  - record.bmp            录制按钮
  - record2.bmp           录制按钮（样式2）
  - usesound.bmp          使用声音按钮
  - Add a catchy title.bmp    标题输入框
  - Add description.bmp       描述输入框

  如果识别不准确，可以自行截图替换对应的模板图片。
  截图时只截取按钮/图标部分，不要包含太多背景。

【设备列表说明】
  - V（勾选）：点击行可以勾选/取消勾选设备
  - 自定义名：在 iMouseXP 控制台中设置的设备别名
  - 设备名：设备的编号名称
  - 型号：手机型号（如 iPhone 11）
  - IP：设备的局域网 IP 地址
  - 刷新设备：重新获取在线设备列表
  - 反选：反转所有设备的勾选状态
  - 全选：勾选所有设备

【常见问题】

  问：提示"未找到 imouse 模块"
  答：双击 install.bat 安装依赖

  问：出现 UnicodeEncodeError 或 surrogates 编码错误
  答：双击 fix_imouse.bat 修复

  问：连接超时
  答：确认 iMouseXP 内核已启动，默认连接 localhost:9911

  问：图像识别失败，找不到按钮
  答：1. 增加"识别等待(秒)"参数
      2. 检查 icon/ 文件夹中的模板图片是否匹配当前 TikTok 版本
      3. 自行截图替换模板图片

  问：install.bat 双击后闪退
  答：右键 install.bat -> 用管理员身份运行

  问：电脑没有 Python
  答：install.bat 会自动检测并安装 Python 3.12，
      如果自动安装失败，请手动双击 python-installer.exe 安装

  问：素材文件夹模式跳过了某些设备
  答：检查子文件夹名称是否和设备的"自定义名"完全一致（区分大小写）
